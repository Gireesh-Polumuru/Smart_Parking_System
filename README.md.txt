#  Smart Parking System

A web-based smart parking system that enables users to book and confirm parking slots in real-time using a visual dashboard and QR code verification.

##  Features

-  Online booking of parking slots
-  Real-time status updates (Green: Empty, Blue: Booked, Red: Confirmed)
-  QR code-based confirmation system
-  10-minute timeout for booked slots without confirmation
-  Secure backend using MongoDB
-  Interactive frontend with slot representation and color codes

##  Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Atlas)
- **QR Code**: `qrcode.js`, custom QR scanner
